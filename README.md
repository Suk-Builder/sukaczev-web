# Sukaczev Web

> Sukačev 视频平台的 Web 前端。

## 这是什么

Sukaczev Web 是 [Sukačev 视频平台](http://suk-baihua.top) 的 Web 前端应用，提供视频浏览、播放、搜索、以及用户管理界面。

## 关联产品

- [Sukačev 视频平台](http://suk-baihua.top) — 主站
- [baihua-workshop](https://github.com/Suk-Builder/baihua-workshop) — 后端服务
- [sukaczev-app](https://github.com/Suk-Builder/sukaczev-app) — 移动端

## 技术栈

TypeScript / React

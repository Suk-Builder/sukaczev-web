这是斐波那契数列的实现：
def fibonacci(n):
    if n <= 1:
        return n
    a, b = 0, 1
    for _ in range(2, n+1):
        a, b = b, a + b
    return b

# 前10项
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")

# 输出: F(0)=0, F(1)=1, 
# F(2)=1, F(3)=2, F(4)=3...
时间复杂度 O(n)，空间复杂度 O(1)

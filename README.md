# Sukačev Web · 前端

Sukačev视频平台的React Web前端应用，提供现代化的视频浏览与交互体验。

## 核心概念

Sukačev开源视频平台的Web入口。采用React 19与Vite构建高性能单页应用，配合Tailwind CSS实现响应式设计，支持桌面端与移动端自适应布局。

## 功能特性

- 视频流瀑布流浏览
- 视频播放器（支持弹幕、倍速、画质切换）
- 用户系统（登录/注册/个人中心）
- 视频搜索与分类筛选
- 实时消息通知
- 深色/浅色主题切换

## 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| React | 19 | UI框架 |
| Vite | 6.x | 构建工具 |
| TypeScript | 5.6 | 类型系统 |
| Tailwind CSS | 3.4 | 样式框架 |
| Radix UI | latest | 无障碍组件库 |
| Recharts | latest | 数据可视化 |

## 快速开始

### 安装

```bash
git clone https://github.com/Suk-Builder/sukaczev-web.git
cd sukaczev-web
npm install
```

### 开发

```bash
npm run dev
```

### 构建

```bash
npm run build
```
